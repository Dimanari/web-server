C_SNEK language manual:


c_snek system functions


print([out_string], varaibles_to_print...) - appends output
sprint([out_string], varaibles_to_print...) - set output
strcmp(string1, string2) - compares strings
set_cookie(cookies, name [,value]) - sets or removes a cookie
query(query_results, variables_for_query_string...)
get_query(query_results, out_string, index)

c_snek system variables:


out_string - the global output string
cookies - the cookie handle
query_results - result handler for database queries
query_prompt - a secondary string for query output

c_snek keywords:


var - declare variable
func - declare function
if - conditional check
else - conditional else
while - conditional loop
return - return from function

operators:


== - compare equals
!= - compare different
>= - compare greater-or-equals
<= - compare smaller-or-equals
> - compare greater
< - compare smaller
= - assign
+ - add
- - subtract
* - mul (*DEREF)
/ - div
& - &REF
() - breacket
{} - code block
, - comma seperator(specific use cases!!!!)
; - Line End
"" - string literal
// - single line comment
/* */ - multi line comment
